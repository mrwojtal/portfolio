PROJECTS

  • Bachelor Thesis - CarDashboard

    This project is a complete implementation of a car dashboard system based on a Raspberry Pi and a Mini R50 tachometer. Vehicle data such as speed, engine RPM, and other parameters is read in real time using an ELM327-based OBD-II module. The diagnostic interface communicates with the Raspberry Pi via Bluetooth, allowing continuous data acquisition. All collected vehicle data is stored in an SQL database. A Python application manages communication with the Mini R50 tachometer over the CAN bus, while a Flask-based web server allows the user to control what is displayed on the tachometer and presents the data in a clear, user-friendly interface.

    • Flask web server serving as a dashboard online interface
    • SQL database as a storage for car data
    • Python aplication integrating bluetooth and CAN communication
    • Reverse engineered Mini R50 tachometer (CAN messages and signals)
    • Created CAN database (.dbc file)
